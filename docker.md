#### redis安装

##### docker command

`docker run --name redis -p 6379:6379 -d redis --requirepass "123456"`

##### docker-compose 

services:
  redis:
    container_name: 'myredis'
    image: 'redis'
    restart: always
    ports:
      - 6379:6379
        command: redis-server --requirepass mypassword
