##### 脚本参数：

```shell
$ script.sh word1 word2 word3
```

脚本文件内部可以使用特殊变量来引用参数。

- `$0`：脚本文件名，即`script.sh`。
- `$1`~`$9`：对应脚本的第一个参数到第九个参数。
- `$#`：参数的总数。
- `$@`：全部的参数，参数之间使用空格分隔。

###### 例子1：

```shell
#!/bin/bash
# script.sh

echo "全部参数：" $@
echo "命令行参数数量：" $#
echo '$0 = ' $0
echo '$1 = ' $1
echo '$2 = ' $2
echo '$3 = ' $3
```

###### 结果1：

```
$ ./script.sh a b c
全部参数：a b c
命令行参数数量：3
$0 =  script.sh
$1 =  a
$2 =  b
$3 =  c
```

例子2：

```
#!/bin/bash

for i in "$@"; do
  echo $i
done
```

##### shift:

`shift`命令可以改变脚本参数，每次执行都会移除脚本当前的第一个参数（`$1`），使得后面的参数向前一位，即`$2`变成`$1`、`$3`变成`$2`、`$4`变成`$3`，以此类推。

##### shell里指令的命令不输出到控制台：

```
>/dev/null 2>&1
```
